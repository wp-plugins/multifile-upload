This plugin allows WP's upload interface to handle multiple files at a time. User can also change the original and thumbnail image sizes.

== Description ==

Multifile Upload plugin adds some features to the WordPress' default upload interface. You will be able to upload more than one file at a time. You can also resize the original image by specifying the size and change the thumbnail size when uploading.

== Installation ==

1. Upload `multifile-upload` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
